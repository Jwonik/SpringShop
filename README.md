# SpringShop
Spring Framework를 활용한 쇼핑몰 프로젝트입니다.
앞서 진행한 Web을 활용한 쇼핑몰 프로젝트를 변환하여 프로젝트를 진행하였습니다.

# Description
+ 개발 기간: 2022.12.28 ~ 2023.01.10

+ 참여 인원: 5명

+ 사용 기술
  + Java
  + Ajax, JQuery
  + Spring Framework
  + Oracle 18c, Mybatis
  
+ 담당 구현 파트
  + Oracle DB 연결 및 테이블 설계
  + 회원가입, 수정, 탈퇴
  + API를 통한 결제
  + 주문, 주문 내역, 주문 상세 내역 관리
  + 관리자 기능 회원, 주문, 환불 관리

# 후기
앞선 Web 프로젝트를 진행하면서 변수명, 클래스명 등을 정하지 않아 발생한 어려움을 보완하고자 했다. </br>
프로젝트를 진행하기에 앞서 규칙을 정하여 진행하여 비교적 수월하게 프로젝트를 진행할 수 있었다. </br>
초반에 Spring Framework와 Mybatis에 있어 오히려 불편함을 느꼈지만, 사용할수록 편리하다는 것을 알 수 있었다. </br>
